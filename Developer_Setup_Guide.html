Digital Farewell Museum - Developer Guide

This developer guide provides step-by-step instructions for extracting, compiling, and running the interactive cinematic college farewell website.

📋 Core Prerequisites

Before launching the server, please verify that the host machine has the following dependencies installed:

  • Node.js: Version 18.0.0 or higher (LTS version highly recommended).
  • NPM: Bundled natively with Node.js to manage asset dependencies.
  • Modern Browser: Google Chrome, Microsoft Edge, Brave, or Safari (supporting standard Web Audio API).

🛠️ Quick Installation Steps

  1. Extract Archives: If the folder was delivered as a compressed .zip file, extract the entire directory to a designated folder on your local drive.
  2. Open Command Terminal: Navigate inside the folder, click the address bar, type cmd (or powershell), and press Enter. Alternatively, open the directory inside VS Code and open the integrated terminal.
  3. Install Node Packages: Run the following command in the terminal to resolve all modular animation packages:
npm install --legacy-peer-deps

Note: The --legacy-peer-deps flag ensures seamless package alignment under React 19 dependencies.

🚀 How to Run the Website

The website includes scripts for both developer sandbox live testing and production-optimized compilation:

1. Active Development Server (Live HMR)

To run the website in hot-reload live mode, execute:

npm run dev

Hold Ctrl and click the local URL generated in your console, or open your web browser and navigate directly to:

http://localhost:5173/

2. Production Optimization Compilation

To build a lightweight, minified, high-performance static site package for direct hosting:

npm run build

To preview the optimized production build locally:

npm run preview

💡 Headphones Recommended: The website contains an offline-synthesized ambient space piano soundtrack. Be sure to turn on audio after clicking "Enter Memory Vault" to experience it!

💎 Architectural Engineering Features

Below is a brief summary of the luxury interaction models programmed into this tribute page:

Web Audio Synth

An offline-synthesized space piano that plays a beautiful, nostalgic chord progression on the fly without heavy external audio files.

Anti-Gravity Starfield

A background particle canvas. Stars drift upward and are gently pushed away from the user cursor in an anti-gravity field.

Memory Vault

Glass cards grouped by category. Features expandable full views, dynamic simulated uploads, and tape-hiss voice echo wave bars.

Farewell Theatre

Chapter selector with 3D tilts. Pulls back theatre curtains on click and projects abstract dreamscapes on canvas screens.
